關於

夜雪羽是從《陰陽師》延伸至《決戰平安京》的陰陽寮，秉持讓寮友以較慢的步調去享受遊戲所帶來的樂趣而創建的休閒型寮辦。

寮辦名為「夜雪羽」的意思是「是夜，落羽飄零如雪」，形容夜裡輕輕飄落的白羽宛如雪花般皎潔且輕盈。命名時因一首《髮如雪》而萌生靈感，故使用了這個名字。

同時，也意味著希望加入寮辦的寮友們能放鬆心情地去享受遊戲，從中體會到遊戲的美好。

感想

時光飛逝，轉瞬之間已然過了兩年之久。

由衷地感謝各位，陪伴寮辦走過了那麼長的道路。

雖說礙於課業，與大家的交流並不頻繁；不過每一次看見各位在聊天室裡的對話，總會不禁感到一絲心情愉悅。

也由衷地感謝願意自發提出聚餐的寮友，多虧了他們悉心的安排，讓參與聚餐的寮友們得以盡興而歸，這讓我感到有些意外、也很是高興。

最後，今年是《決戰平安京》的夜雪羽創寮二週年，祝願寮友們諸事順心、如意，也在這裡說句：二週年快樂！
